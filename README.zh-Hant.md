[SJMC Launcher](https://github.com/UNIkeEN/SJMCL) 是一款最初來源於上海交通大學 Minecraft 社的新一代 Minecraft 啟動器。本倉庫收錄由社群維護的 SJMCL 擴展，涵蓋功能整合、介面增強、生產力工具等各個方面。[>> 擴展開發文件](https://mc.sjtu.cn/sjmcl/dev/extension/)

歡迎透過 PR 新增你的專案，提交前請先閱讀 [CONTRIBUTING.zh-Hant.md](CONTRIBUTING.zh-Hant.md)。

## 示例

- **[Clock](org.sjmcl.clock/)** — 在啟動器介面展示即時時鐘的簡單示例擴展。

## 介面增強

- **[Auto Titlebar](https://github.com/YoshinoHdq/sjmcl-auto-titlebar)** — 滑鼠懸停時顯示標題欄，移出時自動隱藏。
- **[Launcher Theme Enhancer](https://github.com/YoshinoHdq/sjmcl-launcher-theme-enhancer)** — 整合城市天氣、動態背景和粒子特效為一體的一站式啟動器美化擴展。

## 生產力工具

- **[Log Probe](https://github.com/YoshinoHdq/sjmcl-log-probe)** — 分析 Minecraft 錯誤日誌，識別常見問題並提供解決建議。
- **[MineNote](https://github.com/YoshinoHdq/SJMCL-MineNotes)** — 功能比較簡單的小記事本。
- **[SJMCL Pull Request Reminder](https://github.com/sdfwss/sjmcl-pullrequest-reminder)** — 提醒處理 SJMCL 拉取請求的擴展
- **[Streak Calendar](https://github.com/Chang-Yo/StreakCalendar)** — 一個記錄 Minecraft 遊戲活躍度的日曆擴展
- **[ToDo-List](https://github.com/Hongbro886/ToDo-List)** — 一個簡單的 ToDoList 擴展

## 新聞資訊

- **[NewsHomepageSJMCLExtension](https://github.com/Light-Beacon/NewsHomepageSJMCLExtension)** — 在啟動器中查看 Minecraft 簡體中文資訊的擴展

## 娛樂

- **[Daily Meme](https://github.com/zaixiZaixiSJTU/Dailymeme)** — 在啟動器主頁顯示每日 Minecraft 迷因的擴展
- **[MC Daylight](https://github.com/YoshinoHdq/SJMCL-MC_Daylight)** — 即時顯示 Minecraft 伺服器時間與晝夜狀態。
- **[PigHub Random Pig](https://github.com/bowenxiao751-svg/sjmcl-pighub-random)** — 從 PigHub 獲取隨機可愛豬豬圖片，支援首頁卡片、全屏畫廊與自動輪播。
- **[TrophyCraft](https://github.com/YoshinoHdq/SJMCL-TrophyCraft)** — 在啟動器中收集成就，分析你的啟動器使用人格。

## 其他

- **[Footprints](https://github.com/YoshinoHdq/SJMCL-Footprints)** — 自動儲存已刪除的實例為紀念卡片，在首頁展示。
- **[McMSF 漿果服伺服器](https://github.com/Vxiaolong666/mcmsf-SJMC)** — 開放的 Minecraft 伺服器宣傳平台，支援在啟動器中瀏覽、篩選和連接 MCMSF 平台上的各類 Minecraft 伺服器
- **[Server Sentinel](https://github.com/YoshinoHdq/sjmcl-server-monitor)** — Minecraft 伺服器監控面板 - 即時查看線上人數、延遲、TPS 與狀態。
- **[SJMC SMP2 衛星地圖](https://github.com/YoshinoHdq/sjmcl-satellite-map)** — 在啟動器中查看 SJMC SMP2 伺服器的即時衛星地圖，支援多維度切換與 2D/3D 檢視。
