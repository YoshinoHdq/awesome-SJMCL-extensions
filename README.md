[SJMC Launcher](https://github.com/UNIkeEN/SJMCL) 是一款最初来源于上海交通大学 Minecraft 社的新一代 Minecraft 启动器。本仓库收录由社区维护的 SJMCL 扩展，涵盖功能集成、界面增强、生产力工具等各个方面。欢迎通过 PR 添加你的项目，提交前请先阅读 [CONTRIBUTING.md](CONTRIBUTING.md)。
